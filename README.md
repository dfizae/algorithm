## 📌 Commit Message Convention
깃허브 히스토리를 직관적으로 파악하기 위해 아래의 커밋 규칙을 준수합니다.

* **[플랫폼/주제] 문제이름 / 소요시간**
* 예시: `feat: [BOJ] 1260 DFS와 BFS / 40분`

| 태그 | 설명 |
| --- | --- |
| `feat` | 새로운 문제 풀이 및 코드 추가 |
| `refactor` | 기존 코드 수정 및 최적화 (시간복잡도 개선 등) |
| `docs` | README 등 문서 수정 |
